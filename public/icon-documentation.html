Hydra Synthesis - Hyground Documentation

This document describes the purpose and functionality of each icon in the key components of the Hydra visual synthesis application.

EditorView Component

The EditorView component provides the main code editing interface with quick action buttons for Hydra code manipulation.

Main Control Icons

Load Random Example - Loads a random Hydra code example from the examples collection
Special Behavior: Hold Alt key while clicking to generate new code using the generator instead of loading examples
Mutate Current Code - Applies random mutations to the current Hydra code
Special Behavior: Hold Shift while clicking to automatically send mutated code to stage
Send to Stage - Transmits the current code to the stage view for live performance
Generator Settings - Opens/closes the code generator configuration panel
Record/Play Controls - Toggles the visibility of recording and playback controls

InActorPanel Component

The InActorPanel component provides recording, playback, and performance control functionality for live coding sessions.

File Management Icons

Clear Recording Buffer - Removes all recorded content from the current session buffer
Import Recording File - Loads a previously saved recording file into the system
Export Recording File - Saves the current recording to a file for later use
Load Recording into Playback - Transfers recorded content into the playback system

Playback Control Icons

Fast Backward - Rapidly moves backward through the recorded sequence
Step Backward - Moves one step backward in the recorded sequence
Pause - Pauses playback (shown when currently playing)
Play - Starts or resumes playback (shown when paused/stopped)
Step Forward - Moves one step forward in the recorded sequence
Fast Forward - Rapidly moves forward through the recorded sequence
Playback Settings - Opens configuration panel for playback timing and duration settings

HydraStage Component (via StagePanel)

The HydraStage component manages the main performance view and includes controls embedded in the StagePanel for stage-specific functionality.

Stage Control Elements

Edit
Open Code Editor - Opens a new editor window for live coding during stage performance
☐ Fx
Enable Transition Effects - Toggles smooth blending transitions between different Hydra sketches
☐ wgsl
Use WebGPU Shading Language - Enables WebGPU-based rendering for enhanced performance
☐ Quad
Enable Quad Rendering - Activates quad-based rendering mode for specific visual effects
Note: The HydraStage also includes an embedded InActorPanel with all the recording and playback controls listed above.

General Usage Notes

  • Shift Key Modifier: Many actions support holding Shift to automatically send results to the stage
  • Alt Key Modifier: Some actions have alternative behaviors when Alt is held
  • Meta/Cmd Key: The mutate function supports Meta/Cmd for transform changes
  • Conditional Display: Many icons only appear when their functionality is available (e.g., export only shows when recordings exist)
